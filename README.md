#README

#Description
WeatherApp is a web application showing the weather of a city:

- the description of the day's weather
- air humidity
- the wind speed
- the temperature in °C

#Usage
git clone "git@github.com:RemiMaigrot/WeatherApp.git"
cd ./WeatherApp
bundle install
rails server
go to web && type --> localhost:3000
